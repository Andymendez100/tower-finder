LICENSEE LAT & LONG CITY STATE HEIGHT
Description

This application takes the latitude and longitude that you have entered and finds all the cell towers within that city, then the application displays a map that shows your current location as well as creating a marker for all cell towers within the city. It will create a table as well as allowing you to click on the marker of any tower, and display the following information:
The owner of the tower
The coordinates of the tower
City in which the tower is located
State in which the tower is located
Height of the tower
